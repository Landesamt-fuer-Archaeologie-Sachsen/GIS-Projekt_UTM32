# GIS-Projekt\_UTM32

#### für TachyGIS\_arch Version 2

#### Generaldirektion Kulturelles Erbe Rheinland-Pfalz

Diese Projektvorlage ist auf die Nutzung mit TachyGIS\_arch abgestimmt.

Hier sind die Geodatenlayer im Koordinatenreferenzsystem ETRS89\_UTM32 angelegt.

Die hier benutzten Vokabulare orientieren sich an den Anforderungen der Generaldirektion Kulturelles Erbe Rheinland-Pfalz.
